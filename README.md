# About

I'm a first-generation Computer Science graduate currently pursuing my **M.S. in Computer Science** through the **University of Delaware's 4+1 program**.

My primary engineering interests lie in **backend development**, **distributed systems**, and designing software that's scalable, maintainable, and production-ready.

Rather than building AI models from scratch, I enjoy integrating modern AI capabilities—such as **LLM APIs**, **vector databases**, and **Retrieval-Augmented Generation (RAG)**—into practical applications that solve real problems.

Most of my projects are built around the **TypeScript ecosystem**, with **NestJS** serving as my backend framework of choice. I enjoy writing strongly typed, well-structured code and designing APIs that are easy to extend and maintain.
